What's new in 3.0.0

  • Open multiple windows and tabs:
    File -> New window, Close window, Add tab
  • The task template window has been improved
  • You can create sub-contexts, sub-folders and sub-goals:
    In the manage contexts, folders... window, drag and drop items.
  • You can new assign multiple contexts, goals and locations to the same task.
  • Customize the task statuses, the postpone and snooze list:
    Settings -> Lists
  • You can assign a template to a searcher:
    Edit a searcher, then select a template in the list.
    By doing this, if you drag and drop a task on a searcher, the task will be updated according to the template.
