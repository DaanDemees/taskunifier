Enabled:
Define whether the proxy has to be used or not.
Use system proxy:
The current system proxy settings will be used.
Type:
The type of the proxy:
- DIRECT
- HTTP
- SOCKS
Host:
The host of the proxy.
Port:
The port of the proxy.
Login:
The login used to connect to the proxy. This field is optional.
Password:
The password used to connect to the proxy. This field is optional.
