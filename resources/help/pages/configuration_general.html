General

Language:
Language used in TaskUnifier.
You have to restart TaskUnifier in order to apply to selected language.
Date format:
Date format used to display dates.
Time format:
Time format used to display hours.
Use due time:
If this option is not checked, the due dates won't contain any time information.
Ex: 23/01/2011
Use start time:
If this option is not checked, the start dates won't contain any time information.
Ex: 23/01/2011
Minimize to systam tray:
If this option is checked, when you minize TaskUnifier, the software will be minimized in the system tray instead of the system status bar.
Postpone task from current date:
If checked, when you postpone a task, the due date of that task will be used.
If not checked, when you postpone a task, the current date will be used.
Show edit window when a task is added:
If checked, the edit window will appear when you add a task.
Show completed tasks at the end:
Show the completed tasks at the end in the contexts, folders, goals, locations and tags searcher lists.
