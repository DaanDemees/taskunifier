Service:
TaskUnifier can synchronize your tasks will different web services like Toodledo.
Changing the service provider will remove all the data from TaskUnifier!
Conflict choice:
When a task has been edited in Toodledo and in TaskUnifier since the last time you synchronized TaskUnifier, TaskUnifier has to make a choice, keep the Toodledo version of the task, keep the TaskUnifier version of the task or keep the last updated one (recommended).
Keep completed tasks for X days
Only the tasks that are completed for less than X days will be kept in TaskUnifier. The other tasks will of course still be present on Toodledo's website.
Synchronize all data with Toodledo:
The contexts, folders, goals, locations and tasks you updated will be sent to Toodledo. Then all the contexts, folders, goals, locations and tasks will be reloaded from Toodledo.
Remove all the data from the application and replace them with Toodledo's:
Remove all the contexts, folders, goals, locations and tasks from TaskUnifier. Then all the contexts, folders, goals, locations and tasks will be reloaded from Toodledo.
